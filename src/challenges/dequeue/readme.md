In this problem, you are given N integers. You need to find the maximum number of unique integers among all the possible contiguous subarrays of size M.

Note: Time limit is 3 second for this problem.

**Sample Input**
```text
6 3
5 3 5 2 3 2
```

**Sample Output**
```text
3
```
